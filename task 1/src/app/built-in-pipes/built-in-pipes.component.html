1.1 Built-in pipes in Angular

1) DatePipe: formats the date:



Date before format Date after format
{{ currentDate }} {{ currentDate | date: 'yyyy-MM-dd'}}


2) LowerCasePipe: turns a string into lower case:



Result: {{ inputText | lowercase}}

3) CurrencyPipe: formats the currency



Result: {{ inputCurrency | currency:'UAH':'symbol-narrow':'1.2-2'}}

4) SlicePipe: slices the string



Result (characters from 0 to 4): {{ slicedString | slice:0:4}}
